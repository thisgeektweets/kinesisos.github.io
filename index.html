LINUX DISTRO

Kinesis OS was intially built from the ground up on the Linux kernel along with indispensable libraries and apps before adopting the pacman package manager from Arch Linux.

About Kinesis OS

Built by a young determined programmer and developer from a small beautiful island; Mauritius.

Kinesis OS was created based on the guide offered by Linux from Scratch. It aims to provide all the power of Arch linux by using the pacman package manager and ensuring, in a similar way, compatibility, while ensuring a faster and 'easier' setup as well as rolling releases.

Judge for yourself!
